Start a host session or connect as a viewer using a session ID and PIN.

Host

Share your screen, set a required PIN, and optionally enable remote control through the Python agent.

Viewer

Connect with the host-provided ID and PIN. Remote control is only relayed when the host enables it.

Responsible use

Only use this tool on systems you own or where the host has clearly given permission.
